# SAP Fiori Travel Management Application

## Tech Stack

- Frontend: SAP Fiori Elements / SAPUI5 (TypeScript)
- Backend: SAP Cloud Application Programming (CAP)
- Database: SQLite at the moment (PostgreSQL in a bit)
- Deployment: Render

## Features

- Full CRUD operations on Customers & Bookings
- OData V4 service
- Responsive Fiori UI
